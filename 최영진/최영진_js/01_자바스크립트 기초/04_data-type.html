자료형

  • 숫자형(number)
    • 따옴표 없이 숫자로만 표기
    • 계산 작업이 가능한 정수와 실수
  • 문자열(string)
    • 작은 따옴표 또는 큰 따옴표에 둘러싸여 있음
    • 숫자도 따옴표로 묶으면 문자로 인식됨
  • 논리형(boolean)
    • true와 false 2가지 값만 있음
    • 프로그래밍에서 true는 1, false는 0
  • undefined
    • 자료형이 지정되지 않았을 때의 상태
    • 변수 선언만 하고 값을 할당하지 않으면 변수는 undefined 상태
  • null
    • 값이 비어있음
    • 어떤 값이 의도적으로 비어있음을 의미
    • 개발자가 명시적으로 없는 값으로 초기화할 때 사용
  • 오브젝트 타입으로 배열, 객체, 함수가 있다.
