변수(variable)

  • 변수는 데이터를 저장하는 메모리 공간. 즉, 데이터를 저장하는 그릇.
  • 변수를 사용하려면 변수를 구별할 수 있도록 변수에 이름을 붙여주어야 하며 이것을 변수 선언이라고 한다.
  • 변수에는 숫자나 문자열 같은 값뿐만 아니라 객체, 배열, 함수도 저장할 수 있다
  • 변수에는 데이터가 오직 한 개만 저장

변수 var, let, const

ES6 이전의 자바스크립트에서는 var 예약어 만으로 변수를 선언하였다.
ES6에서는 변수를 선언하기 위한 예약어로 let과 const가 추가되었고
되도록이면 var 예약어는 사용하지 않는다.

let : 변수 선언

변수 선언
let 변수명;
변수 선언 후 값을 할당하지 않으면 undefined 상태
값 할당
변수명 = 값;
변수 선언과 값 할당을 동시에 할 수 있다.
let 변수명 = 값;
  • let으로 선언하면 값을 변경할 수 있다. (재할당 가능)
  • let은 같은 이름으로 다시 선언할 수 없다. (재선언 불가능)

const : 상수 선언

변수 선언
const 변수명 = 값;
상수 변수 선언 후 값을 할당하지 않으면 SyntaxError
  • const는 상수 변수
  • 상수란 한 번 값을 할당하면 프로그램 안에서 그 값을 변경할 수 없다.
  • 변하지 않는 값을 변수로 선언할 때 const를 사용한다.
  • const로 선언하면 let과 달리 값을 변경할 수 없다. (재할당 불가능)
  • const로 선언하면 let과 마찬가지로 재선언할 수 없다. (재선언 불가능)

변수 선언 규칙

변수 이름은 의미있게 작성해야 하며
여러 단어를 연결한 변수 이름은 자바스크립트에서는 카멜케이스 방식 권장

  • 자바스크립트는 영어 대소문자 구별한다.
  • 숫자로 시작하면 안된다.
  • 특수문자를 사용할 때는 _ $만 허용한다.
  • 공백 문자를 포함할 수 없다.
  • 예약어를 사용할 수 없다.

변수명 예시

  • let 25now : x
  • const *now : x
  • let $now : o
  • let aaa : x
  • let var : x
