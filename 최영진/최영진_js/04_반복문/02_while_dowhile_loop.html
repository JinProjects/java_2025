while 반복문

초기식; while(조건식) { 실행할 문장; 증감식; }

  1. 초기식
  2. 조건식 확인
  3. 조건식 결과가 true면 { }안에 있는 문장 수행
  4. 증감식
  5. 2, 3, 4 반복하다가
  6. 조건식 결과가 false면 반복문 빠져 나옴

while 반복문은
조건을 먼저 검사하고 코드를 실행하기 때문에
처음 조건이 거짓이면 { }안에 있는 문장을 한 번도 실행하지 않는다.

do - while 반복문

초기식; do { 실행할 문장; 증감식; } while(조건식);

  1. 초기식
  2. { }안에 있는 문장 수행
  3. 증감식
  4. 조건식 확인
  5. 조건식 결과가 true면 { }안에 있는 문장 수행
  6. 3, 4, 5 반복하다가
  7. 조건식 결과가 false면 반복문 빠져 나옴

do - while 반복문은 적어도 한 번은 실행한다.
